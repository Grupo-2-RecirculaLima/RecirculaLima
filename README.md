# RecirculaLima ♻️
**RecirculaLima** es una solución tecnológica móvil diseñada para optimizar y formalizar la gestión de residuos sólidos y el reciclaje en Lima Metropolitana, alineándose con las normativas del MINAM.

##  Segmento Objetivo 
Nuestra plataforma está diseñada principalmente para:
1. **Ciudadanos:** Pueden programar la recolección de sus residuos reciclables, localizar centros de acopio cercanos y hacer seguimiento en tiempo real.
2. **Recicladores:** Optimizan sus rutas de recolección diaria, reciben solicitudes directas de los ciudadanos y registran el material recolectado.
3. **Administradores de Centros de Acopio:** Gestionan el inventario de materiales recibidos, coordinan con las asociaciones de recicladores y generan reportes de impacto ambiental.
---

## Principales Características que ofrece RecirculaLima
 **RecirculaLima** ofrece las siguientes funcionalidades y secciones clave:

*   **Geolocalización y Rutas Óptimas:** Mapeo en tiempo real de puntos de reciclaje y cálculo de rutas eficientes para los recicladores.
*   **Programación de Recojo:** Sistema de alertas y agenda para coordinar recolecciones domiciliarias de forma segura.
*   **Gamificación y Recompensas:** Incentivos para el ciudadano mediante un sistema de puntos canjeables por beneficios locales.
*   **Gestión de Inventario:** Panel de control (*dashboard*) para centros de acopio con métricas de kg de plástico, vidrio, papel, etc.
*   **Historias de Usuario Robustas:** Desarrollo guiado por criterios de aceptación bajo el enfoque *Behavior-Driven Development* (BDD / Gherkin).
